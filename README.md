# keep-project-focus

A Codex skill for keeping one authoritative active project in a multi-project dashboard or AI factory workflow.

## What it captures
- Durable `focusedProjectId` style state
- Correct create/switch/delete behavior
- `Next Action` regeneration from the real focused project
- Proposal id remapping during formalization

## Install
1. Clone or download this repository.
2. Copy the repository folder into `~/.codex/skills/keep-project-focus`.
3. Start a new Codex session.
